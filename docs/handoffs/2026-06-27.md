## Rollout watch — world threshold cache (#338)

The cache **has started warming** (first `alerts`/`both` run after the #338 merge was
the 2026-06-26 21:20 UTC run): `world_threshold_cache.json` now exists with
**cached_count 6** (Madrid, Sevilla, Lyon, Jacobabad, Kuwait City, Mecca). But:

- **That first run reported `degraded` (`sat:True`).** The warm path saturated the
  Open-Meteo **archive** endpoint after ~7 cities. The `OpenMeteoBudget` models the
  general 600/min **weight** limit; the archive endpoint enforces a *separate,
  stricter per-minute request-count* limit the budget doesn't model. So each
  bootstrap run will likely saturate (→ `degraded`) until the cache is nearly warm.
- **Telemetry location correction:** the new world metrics land in
  `source_health.open_meteo_extreme_signals.runs[].error` (the note string:
  `world[cached:X/596 cov:Z warm:A/Bf sat:…]`), **not** in `run_history` details as
  the 2026-06-26 handoff said. Read cached_count/sat there.

### ⚠️ PENDING — world-half eval-gating defect (found this session, not yet fixed)
In `_run_world_cached_half`, the forecast/evaluation loop is gated on `not saturated`
([open_meteo.py](/Users/andrewpuschel/Documents/Claude/theheat/src/orchestrator/sources/open_meteo.py),
the `while i < len(cached_cities) and not saturated` loop). Warming runs **first** and
saturates the archive endpoint, so on every bootstrap run the **evaluation phase is
skipped entirely** (`cov:0.0`) — the world half caches cities but evaluates **none**
for records during the whole ~2-week warm-up, even ones breaking a record.
**Recommended fix:** evaluate already-cached cities **first** (the cheap forecast
hot-path, different endpoint), **then** warm stale cities — so archive saturation can
never block evaluation; and/or cap `WORLD_WARM_BUDGET` to the archive's real
request-count limit so it stops self-saturating. Touches the world-cache run path →
**codex review per the standing rule**. (Largely moot for the European heatwave,
which reganom now covers, but it's a real blind spot for non-US station records.)
