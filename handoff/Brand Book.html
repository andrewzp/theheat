§01

Wordmark

Inter SemiBold, set tight (−0.022em). The wordmark is a masthead, not a logo — it sits at the top of the publication, sets type for the readings below it, and gets out of the way.

Construction

  • Wordmark. Inter SemiBold (600). Tracking −0.022em. Capital T & H, lowercase otherwise. Never all-caps. Never italic.
  • Mark. Thermometer + accent mercury column & bulb. Mark height = wordmark cap-height × 1.6.
  • Gap. Mark-to-wordmark = wordmark cap-height × 0.55.
  • Clear space. Equal to bulb diameter on every side. Nothing else can enter.
  • Min size. Lockup ≥ 80px wide / 0.85" print. Below that, the mark stands alone.

Outline file

handoff/svg/wordmark-outlined.svg ships glyph paths — renders correctly without Inter loaded externally. Same for lockup-outlined.svg and lockup-mono-outlined.svg.

The accent contract

  • One accent. #C2410C. Integer + decimal both ride it — size and weight create the hierarchy, not a second hue.
  • Unit stays neutral. °C, ppm, MW in ink-2.
  • +ΔC callouts in accent.
  • One accent per surface. If you find yourself adding a second hue, the layout is wrong.
  • No gradients, no glows.

§04

Mark

Thermometer with an accent mercury column rising from the bulb. The only mark that appears next to the wordmark; the rest of the icon kit (degree, isotherm, station-pin) is semantic, never identity.

400
200

Sub-24px rule

  • At < 24px, the thin mercury column is dropped — the bulb stays accent so the thermometer still reads. The two-color column is illegible at favicon sizes.
  • Mono fallback. When a single ink is required (print, embossing, mono channels), the mark reduces to a solid silhouette. See mark-thermometer-mono.svg.
  • Reverse. On dark surfaces — the default avatar — the glass becomes Paper; the mercury stays accent. See mark-thermometer-reverse.svg.

Icon kit (semantic, not identity)

Thermometer = mark. Degree, isotherm, station-pin, mercury-bulb stay in handoff/svg/ as semantic helpers used inline in the dashboard, sidebar lists, and tweet meta. Never substituted for the mark.

§05 · LOCKUP
POSITIONS · CLEAR-SPACE · MIN SIZE · LIGHT + REVERSE
§05

Lockup

Mark + wordmark, locked horizontal. There is no vertical lockup. There is no stacked lockup. The masthead reads as a line.

LIGHT — DEFAULT
REVERSE — INK FIELD

Rules

  • Position. Mark left, wordmark right. Always. Bulb sits below x-height baseline.
  • Clear space. Equal to bulb diameter (mark-height × 0.29) on every side.
  • Min size. Lockup ≥ 80px / 0.85". Below that → mark only.
  • Mono. lockup-mono.svg — ink only, accent dropped. Use on second-color print runs and embossed surfaces.
  • Outlined. lockup-outlined.svg, lockup-mono-outlined.svg — glyphs as paths, no font dependency.

§06 · ACCENT RULE
ONE-PAGE CONTRACT · OWNED VS. UNOWNED SURFACES
§06

The accent rule

Accent (#C2410C) is the only color outside the neutral palette. It earns its place once per surface, on a number that matters.

Where accent applies

  • The headline reading. Integer + decimal = accent. Unit = ink-2.
  • The delta line. “+1.4°C above prev. record” in accent.
  • The avatar bulb. The single dot of color in the mark.
  • The dashboard headline. Inside each draft body — same rule, just inside the editor.

Where accent doesn't

  • Chrome, badges, dividers, status pills, source lines, mastheads, page numbers, scores.
  • Backgrounds. Ever.
  • Body text or links.
  • Multi-accent rows. If two readings appear on one surface, only the lead one earns the accent.

The Twitter timeline caveat

Inside the timeline, tweets render as plain text. Twitter strips the typographic system: no integer/decimal contrast, no tabular figures, no accent color. The number reads in body weight, body size, body color.

Owned surfaces (avatar, banner, OG card, screenshots, the operator dashboard) keep the brand. The feed itself is a plain-text channel — design copy that survives without the system. If the post needs the typography to land, the post is wrong.

Owned vs. unowned

OWNED — ACCENT RENDERS
  • Avatar & banner
  • OG / share card
  • Screenshots, embeds, quote-cards
  • Operator dashboard
  • Brand book, usage guide, web
UNOWNED — PLAIN TEXT
  • Twitter timeline body
  • Threads, replies, quote-tweets
  • Mentions, search results
  • RSS, push notifications

Workflow rules

  • Impact-beat gate. Drafts blocked if date-anchored without a human-impact beat.
  • Tiered queue. Extreme · all-time-high · CO₂ · wildfire · severe-weather · outlook.
  • Score ≥ 70 to surface for posting; below that, edit or kill.
  • Daily quota. 24 posts max. Silence is on-brand.
