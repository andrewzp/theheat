Brand fidelity

Masthead, dateline, source-line, monospace meta, single-accent contract: identical to the public surfaces. The dashboard is a private edition of the same publication, not a different product.

Accent rule

Accent (#C2410C) renders only on the headline reading inside each draft body — the integer + unit. Never on chrome, badges, scores, or status. If a draft has no number, no accent.

Impact-beat gate

Each row carries an ✓/⚠ footer line: drafts are blocked if they're date-anchored ("hottest since 1998") without a human-impact beat. The post button greys out until the warning clears.
